# 🎮 Mini Games Collection

Коллекция из 6 простых, но увлекательных браузерных мини-игр. Идеально подходит для отдыха и тренировки реакции, логики и внимания.

## 📋 Список игр

| № | Игра | Описание |
|---|------|----------|
| 1 | **Угадай число** | Компьютер загадывает число от 1 до 100, а вы пытаетесь его угадать. После каждой попытки подсказка: больше или меньше. |
| 2 | **Простая арифметика** | Решите случайный пример на сложение, вычитание, умножение или деление. Проверьте свои вычислительные навыки. |
| 3 | **Переверни текст** | Введите любую фразу – программа покажет её задом наперёд (реверс строки). |
| 4 | **Простая викторина** | Ответьте на 5 вопросов на общую эрудицию. Узнайте свой результат в процентах. |
| 5 | **Камень, ножницы, бумага** | Классическая игра против компьютера. Выбирайте один из трёх вариантов и смотрите, кто победил. |
| 6 | **Генератор случайных цветов** | Нажмите на кнопку – фон страницы изменится на случайный цвет. Отображается HEX-код текущего цвета. |

## 🚀 Запуск проекта

Проект полностью статический – не требует установки сервера или дополнительных зависимостей.

### Локальный запуск
1. Скачайте или склонируйте репозиторий.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge).
3. На главной странице нажмите «Поехали!» или кликните по карточке любой игры.

### Альтернативный способ (если не работает открытие через file://)
Некоторые браузеры блокируют выполнение JavaScript при открытии `file://`. В таком случае:  
- Установите любой локальный сервер (например, [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) для VS Code) и откройте `index.html` через него.

## 🛠 Технологии

- **HTML5** – семантическая разметка
- **CSS3** – адаптивный дизаин, флексбоксы, гриды, анимации
- **JavaScript (ES6)** – игровая логика, обработка событий, работа с DOM
- **Google Fonts** – шрифт Montserrat

## 📱 Адаптивность

Стили оптимизированы под разные устройства:
- Десктопы (ширина ≥ 1024px)
- Планшеты (768px – 1024px)
- Мобильные телефоны (≤ 493px)

На маленьких экранах карточки игр и блоки с описанием меняют расположение, размер шрифтов и отступы для удобного взаимодействия.
